# Relay v0.2.1

Relay v0.2.1 is a patch release focused on better run observability, cleaner `watch` output, and keeping the end-to-end harness aligned with the streamed log contract.

## What's New In v0.2.1

- Relay runners now stream stdout and stderr directly into per-run log files while the run is in progress.
- `relay watch` now surfaces live agent activity from those log files and reads them incrementally so repeated polls stay stable and prompt-cache-friendly.
- `relay pipeline add` can auto-detect an available local runner (`codex` or `claude`) when `--agent-runner` is omitted, and the bundled skill guidance has been updated to match.
- The repository E2E test doubles now emit the same stdout, stderr, and final-message artifacts as the real runners, so watch-related E2E coverage matches the current runtime contract.
